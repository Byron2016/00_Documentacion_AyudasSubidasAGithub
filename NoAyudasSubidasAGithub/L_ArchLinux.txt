- CONCEPTOS
	- SHELL
		- sHELL es el intérprete de la línea de comandos (command-line interpreter CLI)  Pueden server	
			- Interactivo: prompt más el campo de texto donde se ingresan los comandos.
			- No Interactivo: un script inicializa un shell no interactivo 
			interpreter.
			
		- Shell x defecto 
			- echo $SHELL 
		- Shell actual 
			- echo $$
			- cat /proc/xxx/comm (xxx es el id que salió del comando anterior)
			
			- cat /proc/$$/comm
			
	- TERMINAL 
		- Es un EMULADOR de terminal. Que imita el comportamiento del hardware. Y dentro de este se puede ver el SHELL corriendo.
		- Es la interfase al SHELL
